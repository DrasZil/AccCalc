AccCalc v10.0.0

Library-Driven Curriculum Expansion

Version 10.0.0 adds original calculators, lessons, Smart Solver hooks, OCR routing, and workpaper templates inspired by major accounting, financial statement analysis, statistics, retail, and management-services topic families.

New Tools

  • /accounting/dupont-analysis: DuPont ROE Analyzer.
  • /accounting/earnings-quality-analysis: Earnings Quality and Accruals Analyzer.
  • /statistics/confidence-interval: Confidence Interval Helper.
  • /operations/retail-markup-markdown: Retail Markup and Markdown Planner.
  • /finance/capital-rationing-prioritizer: Capital Rationing Prioritizer.
  • /far/provision-expected-value: Provision Expected Value Planner.
  • /afar/franchise-revenue-workspace: Franchise Revenue Workspace.

Architecture

Routes are registered in src/App.tsx, described in src/utils/appCatalog.ts, powered by shared math in src/utils/calculatorMath.ts, and connected through Study Hub, Smart Solver, OCR routing, and workpapers.

Curriculum Coverage

The release strengthens FAR, AFAR, Cost and Managerial Accounting, Financial Statement Analysis, Statistics, Operations, Retail Pricing, and Strategic/Management Services support.

Validation

The release was validated with npm test and npm run build.
