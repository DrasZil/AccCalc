AccCalc v12.0.0

Focus-First UI Organization Overhaul

Version 12.0.0 reorganizes AccCalc around a single rule: keep the main task visible first and make supporting detail available on demand.

Main Changes

  • Added a shared page-menu pattern for lower-priority route support and orientation content.
  • Reworked shared calculator pages so inputs and results stay primary.
  • Rebuilt the homepage around next-step decisions and delayed discovery.
  • Lightened study-page side references so lessons remain textbook-like and calmer.
  • Added a Workpaper Studio focus mode for cleaner worksheet editing.

Focus-First Rules

Visible now

Inputs, main actions, main outputs, essential warnings, and current workflow state.

Discoverable later

Related tools, deeper study notes, orientation help, supporting references, and grouped browsing.

Shared Surfaces

  • CalculatorPageLayout.tsx now keeps solving surfaces primary and routes support into disclosures and the page menu.
  • ContextualPageMenu.tsx standardizes lower-priority page content across major tool pages.
  • HomePage.tsx now emphasizes quick solve, quick scan, workpapers, continue study, and focused discovery.
  • StudyTopicPage.tsx reduces side-reference dominance.
  • WorkpaperStudioPage.tsx adds a student-friendly focus mode for editing.
