AccCalc v10.1.0 Documentation

AccCalc v10.1.0 is a coverage-matrix completion release for calculator and education-layer breadth. It focuses on topics that were still partial, lesson-heavy, or weakly connected in the current app.

New Curriculum Coverage

  • Cost and managerial accounting: Segmented Income Statement Analyzer.
  • Cost and managerial accounting: Activity-Based Costing Allocator for cost pools, cost drivers, assigned overhead, and unit product cost.
  • Audit and assurance: Audit Sampling Planner.
  • Operations: PERT Project Estimate Helper and Quality Control Chart Helper.
  • FAR: Quasi-Reorganization Deficit Cleanup, Financial Asset Amortized Cost Schedule, and Investment Property Measurement Helper.
  • AFAR: Corporate Liquidation Recovery Planner and Joint Arrangement Share Analyzer.
  • RFBT: Obligations and Contracts Issue Flow.
  • AIS: Access Control Review Workspace.

Architecture Map

New pages are registered in src/App.tsx, described in src/utils/appCatalog.ts, supported by shared math in src/utils/calculatorMath.ts, and tested in tests/calculatorMath.test.ts.

Study Hub

The v9 textbook-style Study Hub layout remains in place. v10.1 adds completion modules in src/features/study/studyExpansion450.ts so each new tool has reviewer context, common mistakes, worked-case framing, related calculators, scan signals, and next-topic continuity.

Smart Solver And OCR

Smart Solver route scoring now recognizes segment-margin, sampling, PERT, ABC, FAR financial-asset and investment-property measurement, quality-control, quasi-reorganization, corporate-liquidation, joint-arrangement, obligations/contracts, and access-control language. OCR route rules identify the same topic-family signals from scanned or pasted text.

Workpapers

New templates support segment reporting, audit sampling, PERT, liquidation/quasi-reorganization, combined RFBT/AIS reviewer work, ABC and quality-control support, FAR asset measurement, and joint-arrangement schedules. They are registered in src/features/workpapers/workpaperTemplates.ts.

The hardening follow-up improves Workpaper Studio by deferring live preview evaluation, adding a narrow-screen edit dock, labeling the editable grid region, and centralizing frozen-row/column offsets with CSS variables.

Computation Hardening

  • The confidence interval helper now explains and computes z intervals when population standard deviation is known and t intervals when standard deviation is estimated from the sample.
  • The capital rationing helper now compares classroom profitability-index ranking with exact project-combination search for independent indivisible projects.
  • New shared helpers cover ABC overhead assignment, amortized-cost financial assets, investment-property measurement, joint-arrangement share schedules, and three-sigma quality-control limits.

Maintenance Notes

When adding a future curriculum topic, wire it through route registration, app catalog, Smart Solver, OCR routing, Study Hub, workpapers where useful, and tests. Avoid hidden one-off pages.

Validation

The v10.1 release includes regression tests for shared math, search, workpaper templates, and solve-target suggestions. Final handoff validation uses npm test, npm run build, and a bounded npm run dev probe.
