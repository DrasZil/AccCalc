AccCalc v12.3.0

Smart Solver And OCR Precision

Version 12.3.0 hardens the decision pipeline around topic-first routing, route-aware extraction, selected-route consistency, and structure-first OCR parsing.

Main Changes

  • Classifies topic families before route-specific extraction and prepared-input building.
  • Adds route-family gating so distinctive accounting topics outrank generic finance routes.
  • Keeps Smart Solver UI summaries aligned to the same selected-route state.
  • Preserves comma-separated classroom amounts so OCR and prompt extraction do not collapse values.

Routing And Confidence

Topic-first routing

Route families are ranked before extraction so generic vocabulary cannot dominate the pipeline too early.

Confidence that means something

Confidence now reflects topic fit, contradiction pressure, missing fields, and warning states instead of token overlap alone.

OCR Direction

  • OCR recommendations can use Smart Solver topic-family analysis when it beats generic OCR hints.
  • Route-aware structured fields stay cleaner before autofill or review.
  • Low-confidence OCR outputs remain review-first instead of pretending to be solved.
