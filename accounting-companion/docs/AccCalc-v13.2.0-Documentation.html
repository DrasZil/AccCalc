AccCalc v13.2.0 Documentation

AccCalc 13.2.0 adds a focused onboarding and tutorial layer, plus cleanup for educational-use boundaries and generated test artifacts.

Tutorial System

  • First-time tutorial for Home, navigation, search, Smart Solver, calculators, Study Hub, Workpapers, and Settings.
  • Returning-user quick tour prompt for the first open after this release.
  • Replay controls in Settings for the full tutorial, quick tour, and tutorial state reset.
  • Responsive spotlight and coach-card behavior across mobile, tablet, laptop, and desktop.

Action-Aware Logic

The tutorial can complete steps from real route changes, navigation/search/settings actions, and calculator input interaction. Missing targets and long waits expose fallback guidance so users are not trapped.

Educational-Use Notices

Tax, audit, legal/RFBT, governance, and AIS reviewer pages now show contextual notices explaining that the pages are educational support and not current professional, legal, tax, audit, or organizational guidance.

Generated Artifacts

Test output now compiles into node_modules/.cache/accalc-test-dist. The old .test-dist/ tree is ignored and removed from the Git index.

Validation

  • npm test
  • npm run build
  • bounded dev-server probe when practical
