# AccCalc 2.8.0 Release Notes

## Summary
AccCalc 2.8.0 is a formula-intelligence release. The app now supports safe solve-for-any-variable flows across a broad set of core calculators while also adding stronger statement-analysis and capital-budgeting workspaces.

## Headline additions
- shared solve-for-any-variable engine
- adaptive solve-target UI on core formula pages
- common-size income statement
- common-size balance sheet
- horizontal analysis workspace
- working capital and operating-cycle workspace
- capital budgeting comparison workspace

## Product impact
- faster assignment support for "solve for the missing value" problems
- stronger reporting-analysis coverage for accounting courses
- more consistent explanations, validation, and result structure
- better Smart Solver routing for reverse-solve phrasing

## Safety notes
- reverse solve is only enabled where the math is stable and explainable
- schedule-heavy tools still use structured workflows instead of fake "solve anything" behavior
- offline support remains truthful: local tools are usable after the release caches successfully
