AccCalc v12.4.0

Compact Personalization And Theme System

Version 12.4.0 expands the theme families, keeps Appearance compact by default, and applies personalization state earlier so customization feels cleaner and more reliable.

Main Changes

  • Added Butter, Moss, Palm, Guava, Sunset, Sangria, Seabreeze, Lagoon, and Odyssey while keeping Classic.
  • Rebuilt Settings Appearance around a compact summary and a discoverable gallery.
  • Applied theme, family, contrast, motion, and meta theme-color before React mounts.
  • Added safe migration from older family names to the new palette-led set.

Palette Families

Warm families

Butter, Guava, Sunset, and Sangria bring warmer editorial and coral-led atmospheres.

Cool families

Seabreeze, Lagoon, and Odyssey bring calmer chrome, stronger analytical contrast, and more premium depth.

Appearance IA

  • Mode switching stays visible first.
  • Current theme summary stays visible first.
  • Quick family switching happens in a compact strip.
  • Full gallery browsing is secondary and on demand.

Reliability

  • Theme state persists through the shared app settings store.
  • Legacy stored families map forward safely.
  • Pre-mount theme application reduces flicker during startup and reload.
