AccCalc v8.0.0 Documentation

This handoff package documents the final v8.0.0 system state after the academic-expansion, theme-personalization, and discovery-integration release.

Release Summary

v8.0.0 adds nine new academic routes across managerial accounting, FAR, AFAR, taxation, and operations, then wires them through search, Smart Solver, OCR routing, study support, and workpaper templates. The release also adds a persistent multi-family theme system with pre-paint restoration.

Major Academic Additions

Managerial

Sales Volume Variance and Sales Mix Variance extend variance support beyond budget execution and into sales-side interpretation.

Operations

Safety Stock Planner links forecasting and EOQ workflows to reorder-point buffer policy.

FAR

Dividend Allocation supports preferred-share priority, dividend arrears, and common dividend per share.

Taxation

Estate Tax Helper, Donor's Tax Helper, and Documentary Stamp Tax Helper broaden classroom tax support with visible assumptions.

AFAR

Consignment Settlement and Branch Inventory Loading broaden special-topic support for consignments and branch-home-office review.

Personalization

Classic, Ocean, Slate, Rose, Blossom, Lavender, Sunset, and Emerald theme families now persist across relaunches.

Architecture

  • src/App.tsx registers routes and page entry points.
  • src/utils/calculatorMath.ts is the shared math source of truth.
  • src/utils/formulaSolveDefinitions.ts defines solve targets, notes, warnings, and formatted results.
  • src/utils/appCatalog.ts centralizes route metadata, aliases, and category surfacing.
  • src/features/smart/* powers Smart Solver routing, field extraction, and solve-target hints.
  • src/features/scan-check/services/ocr/ocrRouting.ts handles OCR-based route suggestion patterns.
  • src/utils/formulaStudyContent.tsx and src/features/study/studyExpansion450.ts supply calculator-linked and reviewer-level learning content.
  • src/features/workpapers/workpaperTemplates.ts registers workbook starter templates.

Theme System

The theme system now includes a mode and a family. Settings are stored in src/utils/appSettings.ts, family metadata lives in src/utils/themePreferences.ts, CSS token overrides live in src/index.css, and index.html restores the last saved theme before paint.

Maintenance Guidance

  • Add shared math before route UI.
  • Keep tax assumptions centralized in src/utils/taxConfig.ts.
  • When adding routes, update catalog, Smart Solver, OCR, study support, workpapers, and tests together.
  • Validate with npm test and npm run build.

Known Remaining Limits

  • Some production chunks remain large, especially WorkpaperStudioPage, appCatalog, main, and formulaSolveDefinitions.
  • Tax helpers remain classroom support tools, not legal or filing systems.
  • The catalog is much broader, but future category-hub refinement would still improve large-scale browsing.
