# AccCalc v11.0.0 Release Notes

AccCalc `11.0.0` is the strongest platform upgrade so far. It focuses on product quality, information architecture, weak-track completion, and premium-feel usability rather than raw route inflation.

## Major Product Improvements

- Added shell-level route context rails so users can see the current curriculum area, route surface, track coverage, and suggested next steps.
- Reworked homepage surfacing with curriculum snapshots, route-surface lanes, and clearer track-strength visibility.
- Strengthened Study Hub browsing with a module-shelf section that makes track coverage and learning continuity easier to scan.
- Improved settings with richer theme-family showcase cards while keeping the saved preference and contrast logic intact.

## New Curriculum And Tool Additions

### AIS / IT Controls

- `Business Continuity Planner`
- `AIS Business Continuity and Recovery Governance` lesson module

### Governance / Ethics / Risk

- `Control Environment Review`
- `Governance Control Environment and Ethical Escalation` lesson module

### RFBT / Business Law

- `Defective Contracts Classifier`
- `RFBT Defective Contracts, Remedies, and Enforcement` lesson module

### Strategic / Integrative

- `Business Case Analysis Planner`
- `Strategic Business Case Analysis and Integration` lesson module

Each addition is wired through routes, app catalog, search, Smart Solver, OCR routing, Study Hub, and workpaper templates.

## Workpaper 2.5 Follow-Through

- Workpaper Studio now surfaces workbook context more clearly while editing.
- Active sheets now show stronger context around template source, workbook topic, and next-step route suggestions.
- New assignment-friendly templates were added for continuity/governance, defective contracts, and business-case planning.

## Shared Reliability Improvements

- Added shared math helpers for continuity readiness, control-environment review, and business-case scoring.
- Added regression coverage for those new helpers.
- Kept student-trust behavior explicit: these are structured academic tools, not hidden professional-certification engines.

## Validation

Final validation for `11.0.0` uses:

- `npm test`
- `npm run build`
- a bounded `npm run dev` probe
