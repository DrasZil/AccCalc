AccCalc v11.0.0 Documentation

AccCalc v11.0.0 is a platform-level upgrade focused on stronger shell structure, clearer navigation, richer Study Hub continuity, better weak-track completion, and more connected Workpaper flows.

What Changed

  • The homepage and shell now surface curriculum snapshots, route-surface context, and clearer next-step suggestions.
  • Study Hub now includes a stronger curriculum shelf for track-level lesson browsing.
  • New tools were added for AIS continuity, governance control environment, defective contracts, and strategic business-case analysis.
  • Matching lessons, Smart Solver routes, OCR routing patterns, app-catalog metadata, and workpaper templates were added for the same weak tracks.
  • Workpaper Studio now shows a clearer workbook context panel while editing.
  • Appearance settings now include richer theme showcase previews.

New Completion Additions

  • AIS: Business Continuity Planner
  • Governance: Control Environment Review
  • RFBT: Defective Contracts Classifier
  • Strategic: Business Case Analysis Planner
These additions were intentionally chosen because those tracks were still relatively weak or fragmented compared with the stronger FAR, managerial, and tax families.

Study And Discovery Architecture

AppLayout.tsx, HomePage.tsx, StudyHubPage.tsx, and appExperience.ts now work together to show curriculum context, route type, and next-step guidance more clearly.

Workpaper 2.5 Follow-Through

WorkpaperStudioPage.tsx now keeps workbook topic, active sheet, template source, and related routes visible while editing. New templates support continuity/governance, defective contracts, and business-case planning.

Shared Calculation Helpers

  • computeBusinessContinuityReadiness
  • computeControlEnvironmentStrength
  • computeBusinessCaseScore

These are covered by regression tests and are intentionally framed as academic support helpers, not hidden professional certification engines.

Performance Notes

The app remains broad and some surfaces are still heavy, especially appCatalog, Smart Solver, Scan & Check, and Workpaper Studio. This release improves organization and discoverability without attempting a risky rewrite of those large systems.

Smart Solver and OCR should still be treated as guided assistants. If routing or extraction confidence is weak, review and correction remain the safer path.

Validation

The final v11.0.0 handoff state is validated with npm test, npm run build, and a bounded npm run dev probe.
