# AccCalc v13.0.0 Release Notes

AccCalc `13.0.0` is a major academic expansion release focused strictly on calculators, reviewer material, quiz coverage, and integration across the accounting curriculum.

## Main Academic Additions

- Added Conceptual Framework Recognition Helper for element classification, recognition checks, measurement-basis review, and disclosure cautions.
- Added Revenue Allocation Workspace for two-obligation transaction-price allocation, satisfaction progress, recognized revenue, and contract asset or liability signals.
- Added Taxable Income Bridge for accounting income, permanent differences, temporary differences, loss deductions, current tax, and deferred-tax signals.
- Added Audit Evidence Program Builder for assertion risk, evidence reliability, evidence relevance, coverage depth, contradictory evidence, and procedure-intensity guidance.
- Added Target Costing Workspace for market price, target profit, allowable cost, committed savings, and remaining cost-gap analysis.
- Added an integrated v13 workpaper template for mixed revenue, tax, audit evidence, and strategic cost review.

## Learning And Practice

- Added original Study Hub modules and quizzes for conceptual framework recognition, revenue allocation, taxable income bridge, target costing, audit evidence, and integrated practical financial accounting review.
- Each topic links into the relevant calculator or reviewer workspace and includes route-choice practice.
- Quiz explanations emphasize why the correct route fits, not only which answer is correct.

## Book-Inspired Coverage Anchors

The release uses the user's visible book/photo subject signals as priority anchors: Intermediate Accounting 1/2/3, practical financial accounting, income taxation, auditing theory, strategic cost management, bookkeeping/auditing support, and CPA-review style integration.

All new content is original educational material. No book text is copied or transcribed.

## Discovery And Integration

- App catalog/search now surfaces the new v13 routes.
- Smart Solver recognizes conceptual-framework, revenue-allocation, taxable-income, audit-evidence, and target-costing prompts.
- OCR routing now recommends the new pages for scanned textbook problems, notes, and worked solutions.
- Calculator page menus surface linked Study Hub lessons and quizzes through existing related-study logic.
- Workpaper Studio includes a v13 academic review support template.

## Validation
