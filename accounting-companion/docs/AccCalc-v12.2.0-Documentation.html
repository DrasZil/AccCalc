AccCalc v12.2.0

Shell Tightening And Density Control

Version 12.2.0 finishes the remaining viewport-fit work, removes redundant Learning Hub chrome, fixes settings overflow more systemically, and trims repeated-use friction from the shared shell.

Main Changes

  • Finished viewport-safe shell sizing for page menus, settings, navigation, search, and reminder overlays.
  • Reduced Learning Hub ceremony so module shelves and useful lesson paths appear sooner.
  • Reworked settings rows and cards so long text wraps instead of overflowing awkwardly.
  • Deferred hidden disclosure content and reduced repeated card-level subscriptions for smoother use.

Viewport-Fit Rules

Shared shell metrics

Temporary surfaces now reserve top chrome, bottom chrome, and safe-area insets before sizing themselves.

One overlay model

Menus, drawers, and reminder surfaces share the same viewport-portal and scroll-lock foundation.

Learning Hub Direction

  • Shorter header copy and less self-explanatory chrome.
  • Module shelves moved higher so users reach useful study structure faster.
  • Less chip-heavy metadata on repeated topic and module cards.
  • Category summaries stay helpful without crowding the lesson list.

Settings And Performance

  • Repeated settings rows now use wrap-friendly grid layouts.
  • Permission and reminder cards handle long text and actions more gracefully.
  • Hidden disclosure content mounts lazily after first open.
  • Homepage metadata subscriptions are computed once instead of per route card.
